# FileConcat - Python Edition (🇮🇹 Italiano)

## Introduzione
FileConcat è un semplice strumento con interfaccia grafica (GUI) scritto in Python. Il suo scopo principale è unire facilmente più file di testo sparsi in diverse sottocartelle in un unico file chiamato `output.txt`.

## Specifiche Tecniche
L'applicazione permette di selezionare una cartella di partenza e di filtrare i file da includere in base alla loro estensione, supportando nativamente formati come `.txt`, `.md`, `.java`, `.html` e molti altri. Grazie all'interfaccia basata su libreria Tkinter, l'utilizzo risulta immediato e molto intuitivo. Inoltre, il progetto include uno script batch predisposto per l'avvio rapido e automatico dell'ambiente su sistemi Windows. Il software è rilasciato integralmente sotto licenza open-source GNU GPLv3.

# FileConcat - Python Edition (🇩🇪 Deutsch)

## Einführung
FileConcat ist ein einfaches, in Python geschriebenes Tool mit grafischer Benutzeroberfläche (GUI). Es dient dazu, mehrere Textdateien aus verschiedenen Unterordnern mühelos und schnell in einer einzigen `output.txt`-Datei zusammenzuführen.

## Technische Details
Die Anwendung ermöglicht es, ein Startverzeichnis auszuwählen und die einzuschließenden Dateien nach ihrer Dateiendung zu filtern. Formate wie `.txt`, `.md`, `.java`, `.html` und weitere werden nativ unterstützt. Dank der Tkinter-basierten Oberfläche ist die Bedienung sehr intuitiv und benutzerfreundlich. Darüber hinaus enthält das Projekt ein Batch-Skript für einen unkomplizierten und automatisierten Start unter Windows. Die Software wird unter der GNU GPLv3-Lizenz veröffentlicht.

# FileConcat - Python 版 - Python Edition (🇨🇳 Cinese - 中文)

## 简介
FileConcat 是一个基于 Python 编写的带有图形用户界面 (GUI) 的轻量级工具。它的主要功能是轻松、快速地将分散在多个子文件夹中的文本文件合并为一个单一的 `output.txt` 文件。

## 详细特性
该应用程序允许用户选择一个初始文件夹，并根据文件扩展名过滤要合并的文件，原生支持 `.txt`、`.md`、`.java`、`.html` 等多种常见格式。得益于基于 Tkinter 开发的界面，它的操作非常直观且易于上手。此外，该项目还包含一个批处理脚本，专为在 Windows 系统上实现一键快速启动而设计。本软件基于 GNU GPLv3 许可证开源。
